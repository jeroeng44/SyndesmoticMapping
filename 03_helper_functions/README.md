# Helper Functions

Shared MATLAB utilities used by the analysis pipeline.

## Subfolders

- `3D Slicer/`
  Imports point data from 3D Slicer `.json` and `.stl` exports.
- `Anatomical Analysis/`
  Distance calculations on tibia/fibula geometry.
- `ApplyMotion/`
  Applies rigid-body motion to points and geometries.
- `Coordinate Transformation/`
  Builds and composes coordinate transforms.
- `General Matlab Helper/`
  Small utility helpers.
- `Motion Markers/`
  Reads motion-marker `.mat` files and computes residual motion.
