# SyndesmoticMapping

`SyndesmoticMapping` is a MATLAB repository for dynamic tracking of anterior tibiofibular distance during external rotation loading in a cadaveric syndesmotic injury model.

The current repository contains:

- motion-tracking analysis scripts for computing anterior tibiofibular distance from CT segmentations and motion marker recordings
- shared helper functions for coordinate transforms, motion application, fiducial handling, and 3D Slicer imports
- visualization tools for browsing timestep-wise geometry exports and creating GIFs from rendered frames

## Repository layout

```text
01_motion_acquisition/        torque acquisition and calibration-side scripts
02_motion_tracking_analysis/  main MATLAB analysis pipeline
03_helper_functions/          shared MATLAB utility functions
04_visualisation/             app-based viewer and GIF export
setup_paths.m                 adds repo folders to the MATLAB path
```

## What the code does

The main analysis pipeline:

1. loads motion-marker recordings from a single experiment
2. loads CT-derived tibia, fibula, marker fiducials, and the tibial point of interest
3. computes anterior tibiofibular distance with two motion models
4. exports plots, `.mat` geometry snapshots, and per-experiment summary tables

The repository also includes an acquisition-side MATLAB script in `01_motion_acquisition` and visualization tools for inspecting exported timestep-wise tibia/fibula point clouds interactively.

## Requirements

- MATLAB with support for:
  - App Designer (`.mlapp`)
  - `readtable`, `jsondecode`, `uigetdir`
  - `stlread`
  - `rotm2quat` and `quat2rotm`
- CT segmentation exports from 3D Slicer in `.json` and `.stl`
- motion-marker `.mat` files with a `markers` variable matching the structure expected by `GetFolder_MotionMarker_Data`
- a torque CSV file for each experiment

## Setup

Run this once per MATLAB session:

```matlab
setup_paths
```

Then update the neutral config file:

`02_motion_tracking_analysis/config/experiment_registry.csv`

Replace the placeholder paths with your local paths for:

- motion marker input data
- segmentation input directory
- output directory
- torque CSV file

## How to run the analysis

Run a single experiment:

```matlab
setup_paths
results = Run_Main_Experiment01_MeasureAD_vs_Torque("exp01");
```

Run all experiments listed in the registry:

```matlab
setup_paths
summaryResults = Run_All_Experiments_Summary();
```

## Expected experiment inputs

Each configured experiment needs:

- a motion-marker folder containing timestep `.mat` files
- a segmentation folder containing:
  - `Tibia_PoI.json`
  - `Segmentation_Tibia.stl` optional
  - `Segmentation_Fibula.stl`
  - `TesterScrews.json` optional
  - `MotionMarker_<ID>.json` for each CT-visible marker
- a torque CSV file with the configured torque channel

## How to use the visualisation tools

### App viewer

Open `04_visualisation/MotionTrackingDisplay_Syndesmose.mlapp` in MATLAB. On startup it prompts for a folder containing files named like:

- `Fib_Seg_0001.mat`
- `Tib_Seg_0001.mat`
- `Tib_PoI_0001.mat`

Those files are generated by the analysis scripts inside the configured experiment output directory.
